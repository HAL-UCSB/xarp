# XARP
**X**R **A**gent **R**emote **P**rocedures

## Setup

```bash
# library only
uv add git+https://github.com/HAL-UCSB/xarp.git

# agents and MCP
uv add git+https://github.com/HAL-UCSB/xarp.git --optional agents 
```
